## 失败方向速记

| 方向 | 失败原因 |
|------|---------|
| DPM-Solver加速 | 步数不足→score欠拟合→色块; 步数过多→数值崩塌 |
| DDIM | condition_score不注入pred_xstart, 与引导架构不兼容 |
| Hybrid DPM+DDPM | 不同采样器走不同概率流, 不可拼接 |
| fp16 | dtype不匹配, 需autocast重构 |
| N>1多步引导 | 当前代码N为死参数, 从未生效 |
| S_END参数 | 引导函数未使用, 之前37轮调参大量无效 |

## 核心发现

1. **稀疏引导反超全引导** — restorer相邻步输出有方差, 稀疏减少梯度抖动
2. **s_end=0.5找到天然分工** — 前半密集锁面部结构, 后半纯扩散收敛细节
3. **DPM存在最优步数~100** — 太少欠拟合, 太多数值崩塌
4. **瓶颈在步数不在求解器** — 纯DDPM 50步同样崩溃
5. **N/S_END为死参数** — 引导函数未使用, 之前的调参大量无效
